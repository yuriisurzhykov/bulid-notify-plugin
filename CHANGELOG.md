# Changelog 

## [1.0.0] - 2026.03.17

### Added
- Initial plugin scaffold
- WebSocket server with mDNS autodiscovery
- Gradle build result listener via ExternalSystem API
- Kotlin and Java compiler error/warning parser
- Heartbeat mechanism to keep mobile connections alive
- Settings UI: port, service name, heartbeat interval, notification limits
- Support for multiple simultaneous device connections
